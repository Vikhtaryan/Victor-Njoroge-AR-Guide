Project Documentation

  • Purpose: Demonstrates an AR and GPS-powered tourism guide for Mombasa using only semantic, standards-based HTML5 for clarity and accessibility.
  • Best Practices: All content structures are logical and crawler-friendly; headings are explicitly hierarchical for both humans and search engines. Each section employs semantic tags and ARIA roles for ease of navigation via screen readers. No CSS/JS frameworks or non-semantic containers used.
  • Accessibility: Forms are clearly labeled, navigation is ordered for keyboard accessibility, and all regions are programmatically recognizable.
  • SEO: Meta descriptions, clean titles, single h1, and clear internal linking following 2025 standards.
  • Future Development: CSS and JS will be layered for visual style and interactivity, but the project is fully usable and accessible in semantic HTML5 alone.
